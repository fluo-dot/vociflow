
# ⚡️ VociFlow

VociFlow ist eine moderne, professionelle Web-Anwendung zum Vokabellernen, die zu 100 % lokal in deinem Browser funktioniert. Maximale Privatsphäre trifft auf erstklassiges Design.

## 🚀 Live Demo
Die App ist für GitHub Pages optimiert. Du kannst sie direkt im Browser nutzen, ohne etwas zu installieren.

## 🛡️ Privacy First (Local-First)
- **Keine Cloud-Speicherung:** Alle Vokabeln liegen sicher im LocalStorage deines Browsers.
- **Kein Tracking:** Wir sammeln keine Daten. Was du lernst, gehört dir.
- **100% Offline-fähig:** Einmal geladen, funktioniert die App auch ohne Internetverbindung.

## 🛠 Features
- **Modernes UI:** Gebaut mit Next.js 15, Tailwind CSS und Shadcn UI.
- **Lern-Modi:** Wische durch Karteikarten oder trainiere deine Rechtschreibung im Schreib-Modus.
- **Organisation:** Erstelle Lernsets und organisiere sie in Ordnern.
- **Backup & Share:** Exportiere deine Sets als JSON-Datei, um sie zu sichern oder mit Freunden zu teilen.

## 📦 Installation & Deployment
Wenn du VociFlow selbst hosten oder weiterentwickeln möchtest:

1. **Repository klonen:** `git clone https://github.com/fluo-dot/vociflow.git`
2. **Abhängigkeiten installieren:** `npm install`
3. **Lokal starten:** `npm run dev`
4. **Build für Produktion:** `npm run build`

### GitHub Pages Deployment
Dieses Repository enthält eine GitHub Action, die die Seite automatisch veröffentlicht:
1. Gehe in deinem GitHub-Repository auf **Settings** > **Pages**.
2. Wähle unter "Build and deployment" > "Source" die Option **GitHub Actions** aus.
3. Push deinen Code in den `main` Branch.
