# **App Name**: LernFlux

## Core Features:

- User Account Management: Allow users to register, log in, and manage their profiles to secure their study content.
- Deck Creation & Management: Enable users to create new study decks, add titles, and descriptions, and organize their learning material.
- Flashcard Editor: Provide an interface to add, edit, and delete individual flashcards within a deck, defining both the front and back content.
- Basic Flashcard Review Mode: Display flashcards sequentially for study, allowing users to mark cards as 'known' or 'unknown' to track basic familiarity.
- Quiz Mode (Text Input): Offer a simple quiz mode where users can type in answers for cards to test their recall, providing immediate feedback.
- Basic Progress Tracking: Store and display basic user progress, such as which cards have been reviewed and simple scores from quiz sessions for each deck.
- AI Content Refinement Tool: Utilize an AI tool to suggest alternative phrasing, expand definitions, or generate related concepts for flashcard content based on user input, aiding in deck creation.

## Style Guidelines:

- Primary color: Thoughtful and focused blue (#4580C0) to promote a calm learning environment.
- Background color: A very light, desaturated blue (#EDF0F4) for excellent readability and a clean feel in a light theme.
- Accent color: A vibrant yet analogous cyan (#19B2DB) for interactive elements and key highlights, offering good contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern and clear title presentation. Body font: 'Inter' (sans-serif) for highly readable and neutral text in cards and UI.
- Employ clean, minimalistic line icons that convey action or meaning without clutter, supporting the focused learning experience.
- Adopt a clean, spacious layout with ample white space around content, centralizing study elements to minimize distractions.
- Subtle and quick transitions for flashcard flipping and view changes, ensuring a smooth, non-intrusive user experience.
